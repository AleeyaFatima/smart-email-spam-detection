Understanding Probability Scores

How the models convert numerical vector spaces into confidence percentages.

Linear/Log Probability Mapping

Every machine learning algorithm outputs a value representing the calculated distance from a decision hyperplane. In classification, we apply a Sigmoid function (for Logistic Regression) or calibration mapping to convert distance to a score between 0.0 (ham) and 1.0 (spam).

Spam Threshold Boundaries (0.50)

By standard classification design, a score equal to or greater than 50% classifies the query message as Spam. Scores below 50% classify the message as clean Ham.

Confidence Calculations

How SpamShield AI calculates and deconstructs model certainty.

The Absolute Deviation Metric

If a model outputs a probability of 0.50, it means it is completely unsure (50% chance spam, 50% chance ham). Thus, its confidence is 0%. If a model outputs 0.00 (certainly ham) or 1.00 (certainly spam), its confidence is 100%.

Formula: Confidence = |Probability - 0.5| * 2

About SpamShield AI

SpamShield AI is an industry-grade threat detection dashboard and machine learning classifier system.

Developed by Aleeya as a professional portfolio project demonstrating clean architecture, REST API route design, and comparative ML model evaluations across statistical frequency models (TF-IDF) and context neural vector networks (Word2Vec).

Architectural Layers:
• Core Layer: Domain dataclasses agreements.
• Data Pipeline: NLTK text cleaner and splits parser.
• ML Engine: Algorithm executors (LogReg, NaiveBayes, SVM, RandomForest).
• Database Layer: SQLite audit logging registers.
• Web Service Node: FastAPI REST endpoint router.
• Dashboard: Pure HTML/CSS/JS single page application interface.
